# Intro to LangChain - Foundation Course

Working through the [LangChain Academy](https://academy.langchain.com/) Introduction to LangChain foundation course. This repo contains my notes and exercises as I go.

## What's in here

- **mod_1.py** - Module 1 basics: chat models, agents, tool use, and checkpointing using HuggingFace + LangGraph.
- **mod_1_chef_agent** - A small recipe-search agent that uses Tavily to find recipes based on available ingredients.

## Stack

- LangChain + LangGraph
- HuggingFace (DeepSeek-R1 model)
- Tavily (web search for recipes)
